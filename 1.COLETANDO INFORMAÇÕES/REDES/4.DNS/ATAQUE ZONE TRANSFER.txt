O ATAQUE DE TRANSFERENCIA DE ZONA OCORRE QUANDO UM NAME SERVER PRECISA FAZER UMA REQUISIÇÃO PARA ATUALIZAR AS ENTRADAS DNS, E COM ISSO VOCÊ CONSEGUE COLETAR MUITAS INFORMAÇÕES QUE SERIA OS SUBDOMINIOS.
	
	1.ENCONTRAR OS NAME SERVER.
		
		host -t ns seusite.com
		
	2.FAZER A TRANSFERENCIA DE ZONA USANDO UM DOS NAME SERVER.
		
		dig axfr seusite.com @name_server_alvo
		
		EX: dig axfr zonetransfer.me @nsztm1.digi.ninja
		
	3.CASO DÊ ERRO COM UM NAME SERVER, TENTE COM OUTRO.
	
	4.CASO DÊ CERTO, APARECERÁ SUBDOMINIOS ETC...
	
	EXTRA: TEM O DNSENUM QUE TENTA O ATAQUE DE ZONA DE TRANSFERÊNCIA AUTOMATICAMENTE.
