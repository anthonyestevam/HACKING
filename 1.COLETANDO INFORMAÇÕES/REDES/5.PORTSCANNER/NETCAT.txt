O NETCAT É UMA FERRAMENTA DE SCAN,ESCUTA E CONEXÃO DE PORTAS.

	nc -nvlp porta:
	
		FAZ SEU PC ESCUTAR A PORTA SELECIONADA.
		
	nc -v ip_do_alvo porta:
	
		FAZ VOCE SE CONECTAR COM UMA PORTA DE UM DETERMINADO IP E VOCÊ PODE ENVIAR UMA REQUISIÇÃO
		QUALQUER COMO QUALQUER LETRA PARA RECEBER A RESPOSTA E ASSIM PODENDO SABER AS VEZES QUAL SERVIÇO 
		ESTÁ RODANDO.
		
	nc -vz ip_do_alvo porta:
		FAZ UM SCAN PARA SABER SE A PORTA SELECIONADA ESTÁ ABERTA.
		
	nc -vnlp porta > arquivo.txt:
		TUDO QUE VOCÊ ESCUTAR NESSA PORTA SERÁ TRANSFERIDO PARA O ARQUIVO TXT CRIADO.
		
	nc ip_do_destinatario < arquivo.txt:
		MANDA DADOS DO ARQUIVO TXT PARA O IP DO DESTINATARIO.
		
OBS: APÓS USAR UM nc ip_do_site porta_smtp... PODE SE USAR O COMANDO 'VRFY nome' PARA SABER SE EXISTE UM USUARIO COM ESSE NOME.
