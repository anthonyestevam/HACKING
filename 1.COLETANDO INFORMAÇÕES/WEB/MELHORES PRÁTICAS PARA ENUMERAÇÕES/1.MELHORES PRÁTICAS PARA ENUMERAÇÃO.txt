A MELHOR PRÁTICA PARA ENUMERAÇÕES É O USO DE MAIS DE UMA FERRAMENTA
PARA A MESMA AÇÃO, POIS AS VEZES O QUE UMA FERRAMENTA COLETA, A OUTRA
NÃO COLETOU E ASSIM POR DIANTE, ENTÃO É INTERESSANTE QUE TENHA O USO DE 
DIVERSAS FERRAMENTAS PARA A MESMA TASK.
	
	DICA: USANDO A FERRAMENTA ANEW PARA SALVAR OS ARQUIVOS OTIMIZA SEU TEMPO
	PODENDO SALVAR OS OUTPUTS DE TODAS AS FERRAMENTAS NO MESMO ARQUIVO E
	DESCARTANDO DUPLICADOS.

	OUTRA PRÁTICA NECESSÁRIA SERIA ENUMERANDO NOVAMENTE AS MESMAS COISAS, MAS
	COM OS NOVOS DADOS COLETADOS, POR EXEMPLO COLETANDO SUBDOMINIOS DOS SUBDOMINIOS
	UMA FERRAMENTA QUE PODE OTIMIZAR SEU TEMPO EM ALGUMAS FERRAMENTA É O XARGS.
	
	A VALIDAÇÃO DE URLs TAMBÉM É INTERESSANTE POIS ALGUMAS URLs QUE OS CRAWLERS
	PEGAM JÁ NÃO EXISTEM MAIS E ASSIM VOCÊ PERDERIA TEMPO AINDA COM URLs QUE 
	NEM FUNCIONAM MAIS E PARA OTIMIZAR ESSE TEMPO PODE SE USAR UMA FERRAMENTA
	QUE CHECA OS STATUS DE CADA URL, FERRAMENTAS COMO HTTPX OU HAKCHECKURL 
	FAZEM ISSO.
