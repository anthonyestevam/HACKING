O SHADOW CLONE SERVE PARA OTIMIZAR TEMPO QUANDO SE TRATA DE SERVIÇOS ENORMES, UMA OTIMA JUNÇÃO
É UTILIZÁ-LO COM O HTTPX PARA VALIDAR URLs BEM MAIS RÁPIDO QUANDO SE TRATA DE MUITAS URLs.
	
	COMANDO: python3 shadowclone.py -i url.txt -s 300 -o validurls.txt -c 'httpx -l {INPUT} -t 200
	
	O -S É O SPLIT, SERIA COMO SE FOSSE A VELOCIDADE, CASO TENTE RODAR COM UM VALOR COMO 300
	E A FERRAMENTA DÊ ERRO, TENTE DIMINUIR PARA 200 OU MENOS ATÉ FUNCIONAR.
	
	-I É O INPUT OU SEJA O ARQUIVO COM AS URLs NO CASO.
	
	-O É ONDE IRÁ SALVAR A SAÍDA DO SHADOWCLONE.
	
	-C É PARA RODAR A FERRAMENTA QUE SERIA O HTTPX NO CASO, MAS PODE SE USAR OUTRAS, EM SERVIÇOS
	QUE ACREDITE QUE DEMORE COMO O NUCLEI POR EXEMPLO, PARA RODAR MAIS RÁPIDO AS ANÁLISES.
