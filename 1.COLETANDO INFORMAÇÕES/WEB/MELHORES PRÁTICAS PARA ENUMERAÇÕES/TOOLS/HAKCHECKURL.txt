HAKCHECKURL É UMA FERRAMENTA QUE CHECA URLs PARA CHECAR SE ESTÃO FUNCIONANDO
OU SE SÃO INEXISTENTE.
	
	COMANDO PARA CHECAR DIVERSAS URLs: cat subdomains.txt | hakcheckurl
	
	A FERRAMENTA CHECA AS URLs DANDO CÓDIGO 200 PARA AS QUE ESTÃO FUNCIONANDO
	E 404 PARA AS QUE NÃO EXISTEM.
	
	cat subdomains.txt | hakcheckurl | grep -v '404' | awk '{print $2}'
	
	ESSE GREP IRÁ EXLCUIR OS SITES 404 E IRÁ MOSTRAR APENAS OS QUE
	DERAM CÓDIGO 200 E NÃO MOSTRARÁ O CÓDIGO 200, APENAS A URL, FICANDO
	PRONTO PARA SALVAR EM UM ARQUIVO.
