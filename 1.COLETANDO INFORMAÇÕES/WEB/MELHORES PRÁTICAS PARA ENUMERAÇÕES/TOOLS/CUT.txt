CUT É UMA FERRAMENTA QUE PODE SER USADA PARA CORTAR ELEMENTOS.	
	
	EXEMPLO:
		UM ARQUIVO CONTÉM A URL: https://site.com
		
	COMANDO: cut -c 9- arquivo.txt
	
		A URL FICARÁ: site.com
		
		OU SE USAR: cut -c 1-4 arquivo.txt
		
		A URL FICARÁ: http
		
		ESSA FERRAMENTA PODE SER USADA POR EXEMPLO PARA TIRAR O HTTPS DE TODAS AS URLS 
		ENCONTRADAS PARA PODER RODÁ-LAS NO AQUATONE.
