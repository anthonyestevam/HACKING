O GOBUSTER É UMA FERRAMENTA PARA ENCONTRAR DIRETÓRIOS EM WEBSITES.
	
	dir: PARA USAR DIRETÓRIO PARA ENCONTRAR A WORDLIST.
	
	-u: PARA COLOCAR UMA URL ALVO.
	
	-w: PARA USAR UMA WORDLIST PARA ENCONTRAR OS DIRETÓRIOS
	
	EX: gobuster dir -u https://seusite.com -w /usr/share/dirb/wordlists/
	common.txt 
	
OBS: ESSA WORDLIST USADA É DA FERRAMENTA DIRB, MAS PODE SE USAR QUALQUER WORDLIST
