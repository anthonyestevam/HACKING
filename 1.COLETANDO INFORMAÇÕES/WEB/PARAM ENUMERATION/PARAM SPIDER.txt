O PARAM SPIDER É UMA FERRAMENTA PYTHON QUE É USADA PARA ENCONTRAR URLs DE UM DOMINIO QUE TENHA PARÂMETROS.
	
	EX: https://site.com/?search=item	
		
		O SEARCH É O PARÂMETRO QUE PODE SER VULNERÁVEL PARA ATAQUES DE XSS, SQL INJECTION,
		SSTI, CSTI, ETC...
		
	COMANDOS:
		
		python3 paramspider.py -d dominio.com: PARA ENCONTRAR OS PARÂMETROS DE SÓ UM DOMINIO.
		
		python3 paramspider.py -d <dominio> -s subdomains.txt: É USADO PARA ENCONTRAR OS PARÂMETROS DO DOMINIO E DOS
		SUBDOMINIOS QUE TERÃO QUE ESTAR EM UM ARQUIVO TXT.
		
	O PARAM SPIDER PODE SER BAIXADO PELO GITHUB.

	EXTRA: O PARAM SPIDER PODE SER USADO EM CONJUNTO COM A FERRAMENTA DALFOX
		USANDO O PARAM SPIDER PARA DESCOBRIR AS URLs COM PARÂMETROS, SALVÁ-LAS EM UM
		ARQUIVO E USAR O ARQUIVO NO DALFOX.
