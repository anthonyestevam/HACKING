O AQUATONE É UMA TOOL MUITO INTERESSANTE PARA COLETA DE INFORMAÇÕES, ELA CRIA UM ARQUIVO HTML QUE MOSTRA PARA VOCÊ AS TECNOLOGIAS UTILIZADAS NOS SITES ESCANEADO, E MOSTRA OS HEADERS UTILIZADOS NELE.
	
	COMANDO: cat subs | aquatone
	
		 echo site.com | aquatone
		 
		VOCÊ PODE ESCANEAR URL QUE TENHAM ENDPOINTS.
		
		
		APÓS A COLETA, BASTA APENAS ABRIR O ARQUIVO HTML GERADO PELO AQUATONE QUE LÁ TERÁ 
		TODOS OS DADOS.
		
		VOCÊ TAMBÉM PODE USAR IPs NO AQUATONE
		
		EX: cat ips | aquatone
	
		PODE USAR O -threads PARA AUMENTAR A VELOCIDADE DO PROCESSO.
		
		-ports UTILIZADO PARA O AQUATONE FAZER O SCAN EM MAIS PORTAS, TEM AS OPÇÕES
		SMALL, MEDIUM, LARGE E XLARGE. EX: cat urls | aquatone -ports xlarge
