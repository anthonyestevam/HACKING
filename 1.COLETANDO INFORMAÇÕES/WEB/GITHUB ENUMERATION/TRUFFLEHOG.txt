O TRUFFLEHOG É UMA FERRAMENTA QUE PROCURA POR CHAVES DE API, ETC...
	
	COMANDOS: trufflehog github --org <organização>
	
	OUTRA FORMA SERIA USANDO APENAS O COMANDO TRUFFLEHOG.
		
		1.ESCOLHA A OPÇÃO 'SCAN A SOURCE USING WIZARD'.
		
		2.ESCOLHA A OPÇÃO GITHUB
		
		3.INSIRA O NOME DA ORGANIZAÇÃO EM ORGANIZATION.
		
		4.CLIQUE EM NEXT.
		
		5.CLIQUE EM 'RUN WITH DEFAULT'
		
		6.CLIQUE ENTER.
		
	OBS: PARA ENCONTRAR O NOME DA ORGANIZAÇÃO BASTA PESQUISAR POR ELA NO GITHUB E COLOCAR
	O MESMO NOME NO TRUFFLEHOG.
