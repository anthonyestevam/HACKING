UMA FORMA MUITO BOA DE ENUMERAR APIs SERIA USANDO CRAWLERS E DEPOIS UM GREP PARA FILTRAR MOSTRANDO
APENAS URLs QUE TENHAM /api/  /v1/  /v2/  /swagger/ /graphql    ETC...

		1.FAÇA ENUMERAÇÃO DE ENDPOINTS COM CRAWLERS.
		
		2.AGORA FILTRE AS URLs QUE CONTENHAM PARÂMETROS COMO 
		/api/  /v1/ /v2/  /api/v1/  /api/users  
	
		EX: cat urls.txt | grep "/v1/"
		
		3.SE TIVER URLS SALVE ELAS EM UM ARQUIVO.
		
		EX: cat urls.txt | grep "/v1/" >> apis.txt
		
		4.APÓS FAZER TODA A COLETA, VALIDE AS URL USANDO UM HTTPX PARA
		ANALISAR QUAIS OS STATUS CODE DAS URLS.
