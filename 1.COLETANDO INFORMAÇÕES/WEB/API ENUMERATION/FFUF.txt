O FFUF É UMA FERRAMENTA DE FUZZ, PODENDO SER UTILIZADA PARA ENUMERAÇÃO DE APIs.

	COMANDO: ffuf -u http://site.com/FUZZ -w assetnote.txt -mc 200
	
	NO -U SERÁ COLADO A URL DO SITE COM UM /FUZZ PARA A FERRAMENTA ENTENDER ONDE SERÁ
	O FUZZING.
	
	O -W É PARA UTILIZAR A WORDLIST BAIXADA NO ASSETNOTE.
	
	O -MC 200 É PARA APARECER APENAS AS URLs QUE DERAM CÓDIGO 200.
