O PACU FRAMEWORK É USADO PARA FAZER EXPLORAÇÃO EM CLOUD AWS, PRECISANDO DA CREDENCIAL DA CLOUD PARA
FAZER A EXPLORAÇÃO.	
	
	1.INSTALAR E SE CONECTAR NA AWS.
	
	2.ENTRAR NA FERRAMENTA.	
		
		COMANDO: pacu
		
	3.O PACU IRÁ PEDIR UM NOME PARA A SESSÃO QUE SERÁ CRIADA, PODENDO ACESSA-LAS POSTERIORMENTE
	CASO QUEIRA.
		
	4.IMPORTAR A CHAVE CREDENCIAL DO AWS PARA O PACU.
		
		COMANDO: import_keys --all
		
		
	5.EXPLORAR.
		
		COMANDOS PARA EXPLORAÇÃO:
			
			whoami: MOSTRA AS INFORMAÇÕES DO USUÁRIO EM QUE VOCÊ ESTÁ CONECTADO
			
			list: MOSTRARÁ TODOS OS COMANDO DE ENUMERAÇÃO E EXPLORAÇÃO
			
			PARA USAR OS COMANDOS DO LIST, USE O COMANDO 'run'.
				
				COMANDO: run comando_escolhido
				
				EX: run ec2_enum
