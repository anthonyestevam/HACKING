O SCOUT SUITE É UMA FERRAMENTA QUE VERIFICA A SEGURANÇA DE UM EM AMBIENTE EM CLOUD, REPORTANDO TODOS OS PROBLEMAS
ENCONTRADOS, ELE É USADO APÓS VOCÊ CONSEGUIR UMA CREDENCIAL DA CLOUD.
	
	
	1.VOCÊ PRECISA SE INSTALAR E SE CONECTAR NO CLOUD.
	
	2.APÓS ISSO PARA ESCANEAR É SIMPLES.
		
		COMANDO: scout nome_cloud
			 
			 EX: scout aws
			 
			 EX2: scout azure 
			 
	3.ELE IRÁ ESCANEAR, NO FIM DO ESCANEAMENTO SERÁ TUDO SALVO EM UM ARQUIVO HTML.
	
	4.ABRIR O ARQUIVO HTML NO SEU NAVEGADOR, USANDO A EXTENSÃO LIVE SERVER DO VSCODE OU
	AVRINDO UM SERVIDOR HTTP PYTHON OU APACHE PARA VER OS PROBLEMAS ENCONTRADOS.
