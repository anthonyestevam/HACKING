O FRIDA É UMA FERRAMENTA QUE FAZ ANALISE E EXPLORAÇÃO DINÂMICA DE UM APP.
	
		PARA USAR É NECESSARIO INSTALAR A VERSÃO CLIENTE NO COMPUTADOR E BAIXAR A VERSÃO SERVIDOR, DESCOMPACTAR E ENVIAR PARA O DISPOSITIVO.
		
	SOBRE A VERSÃO SERVIDOR:
	
		1.NO DISPOSITIVO ENVIE O ARQUIVO COM ADB PUSH: adb push <server> /tmp
		
		2.ENTRE NO ADB SHELL E ENTRE NO DIRETÓRIO tmp.
		
		3.TRANSFORME O ARQUIVO EM EXECUTÁVEL UTILIZANDO O CHMOD: chmod +x <server>
		
		4.EXECUTE O SERVER PARA RODAR UTILIZANDO ./: ./<server>
	
		5.AGORA ABRA OUTRO TERMINAL PARA SE COMUNICAR.
	
	COMANDO DO FRIDA BÁSICO:
	
		frida-ps -aU: MOSTRA OS APLICATIVOS E OS NOMES DOS DIRETÓRIOS DELES.
		
		
	FRIDA CODESHARE:
	
		UMA FONTE ONDE EXISTEM MUITOS SCRIPTS PARA DIVERSAS FINALIDADES COMO BYPASS DE SSL PINNING OU DE ROOT É O FRIDA CODESHARE.
	
		PARA ACESSA PESQUISE NO NAVEGADOR: frida codeshare.
	
		NELE HAVERÁ DIVERSOS SCRIPTS PARA O FRIDA, BASTA ACESSAR O SCRIPT E COPIAR O CÓDIGOPARA EXECUTAR, SUBSTITUINDO O 'BINARY' PELO NOME DO DIRETÓRIO DO APK.
	
	
	OUTRA FORMA DE CONSEGUIR EXPLORAR MAIS FUNÇÕES DO FRIDA SERIA USANDO O AUXILIO DE UMA IA COMO O CHATGPT.
