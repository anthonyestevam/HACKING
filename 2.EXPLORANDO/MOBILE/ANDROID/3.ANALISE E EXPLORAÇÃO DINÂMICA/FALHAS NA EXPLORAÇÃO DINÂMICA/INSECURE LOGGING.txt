Insecure Logging é uma vulnerabilidade que ocorre quando  o aplicativo salva dados como Login, senha, ou qualquer dados pessoal ou tokens nos Logs do dispositivo.

	1.Vá no terminal e digite: adb logcat. Assim o terminal mostrará os Logs do dispositivo.
	
	2.Vá no aplicativo em algum input que seja para inserir dados ou atualizar dados.
	
	3.Preencha esse input e envie.
	
	4.Vá para o terminal onde está os Logs e veja se os dados que foram inseridos apareceram nos Logs.
