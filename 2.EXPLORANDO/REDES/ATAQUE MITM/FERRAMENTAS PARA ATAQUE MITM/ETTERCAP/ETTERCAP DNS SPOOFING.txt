O DNS SPOOFING É UM ATAQUE MUITO USADO PARA ATAQUE DE PHISHING, ONDE VOCÊ ALTERA A TABELA DE DNS FAZENDO O ALVO
ACHAR QUE ESTÁ ACESSANDO O SITE VERDADEIRO, QUANDO NA VERDADE ESTÁ ACESSANDO SEU SITE CLONADO PARA OBTER 
CREDENCIAIS PRO EXEMPLO.
	
	1.CRIAR UM SITE FALSO.
	
	2.EDITAR O ARQUIVO DE DNS DO ETTERCAP.
		
		1.IR NO TERMINAL
		
		2.EDITAR O ARQUIVO.
			
			COMANDO: nano /etc/ettercap/etter.dns
			
		3.ADICIONAR NO FINAL DO ARQUIVO O DOMINIO ALVO NO DNS COM O 
		IP QUE ESTÁ O SITE FALSO DUAS VEZES, SENDO O PRIMEIRO COM UM 'A'
		ENTRE O DOMINIO E IP E O SEGUNDO UM 'PTR' ENTRE O DOMINIO E IP.
			
			EX:	
				
				seusite.com	A	192.16.23.10
				seusite.com	PTR	192.16.23.10
				
	3.ENTRAR NO ETTERCAP E ADICIONAR O IP DO ROTEADOR COMO TARGET 1
	
	4.ADICIONAR O IP DO ALVO COMO TARGET 2.
	
	5.INICIAR O ARP SPOOFING.
	
	6.INICIAR O DNS SPOOFING.
		
		1.CLICAR NAS TRÊS BARRINHAS.
		
		2.IR EM PLUGINS.
		
		3.CLICAR NA OPÇÃO 'MANAGE PLUGINS'.
		
		4.CLICAR DUAS VEZES NA OPÇÃO DNS SPOOF.
