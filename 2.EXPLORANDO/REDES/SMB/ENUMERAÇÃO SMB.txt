O SMB(SERVER MESSAGE BLOCK) É UM SERVIÇO DE PASTAS COMPARTILHADAS QUE RODA NORMALMENTE NA PORTA 445.
COM ESSA PORTA ABERTA, NORMALMENTE AS VERSÕES QUE RODAM NO SMB É O MICROSOFT-DS OU NETBIOS, NO
LINUX TEM O SAMBA QUE TEM O MESMO FUNCIONAMENTO DO SMB QUE É DO WINDOWS.

	1.VISUALIZAR QUAIS PASTAS COMPARTILHADAS TEM DISPONÍVEIS.
		
		COMANDO: smbclient -L <ip_alvo>
		
	2.LOGO APÓS USAR ESSE COMANDO, O SERVIÇO SOLICITARÁ UMA SENHA, BASTA 
	CLICAR EM 'Enter' COM A SENHA EM BRANCO.
	
	3.SE APARECER AS PASTAS, VOCÊ PODE TENTAR SE CONECTAR EM CADA 
	UMA PARA TALVEZ ENCONTRAR ALGUMA QUE POSSA ENTRAR COM SENHA EM 
	BRANCO.
	
		PARA ENTRAR NA PASTA SE USA O COMANDO: 
		
		      smbclient //<ip_alvo>/<nome_da_pasta>
		      
		      EX: smbclient //10.129.1.12/Workshares
		      
	4.SE PEDIR SENHA, CLIQUE 'Enter' COM A SENHA EM BRANCO, CASO CONSIGA 
	SE CONECTAR USE O COMANDO 'ls' PARA LISTAR O QUE TIVER DENTRO DA
	PASTA E USE 'cd' PARA ENTRAR DENTRO DE UMA PASTA.
	
	5.APÓS VER O QUE ENCONTROU DENTRO, USE O COMANDO 'get' PARA BAIXAR 
	ARQUIVOS. EX: get arquivo.txt
	
ASSIM VOCÊ PODE CONSEGUIR DADOS IMPORTANTES PELO SMB.

	OBS: ESSE PASSO PODE SER USADO NA PÓS EXPLORAÇÃO APÓS VOCÊ CONSEGUIR CREDENCIAIS DE
	UM USUÁRIO, USA-LAS PARA TENTAR COLETAR DADOS IMPORTANTES.
		
		USANDO O COMANDO: smbclient -L <ip_alvo> -U <nome_usuario>
		
		ESSE COMANDO SERVE PARA USAR O USUÁRIO QUE VOCÊ DESCOBRIU E COLOCAR
		A SENHA QUE VOCÊ DESCOBRIU, LOGO ASSIM QUE PEDIR.
