O QUE PODE OCORRER MUITO É ROTEADORES COM FALTA DE CONFIGURAÇÃO, PODENDO ASSIM VOCÊ CONFIGURAR UM ROTEADOR
COM SENHAS PADRÕES DE FÁBRICA, ISSO PODE OCASIONAR PROBLEMAS DE DISPONIBILIDADE, PORQUE VOCÊ COMO
INVASOR PODE ALTERAR A SENHA DO ROTEADOR FAZENDO AS PESSOAS PERDEREM O ACESSO A ELE.
	
	PARA ACESSAR O ROTEADOR BASTA COLOCAR O IP DELE NO NAVEGADOR, ABRINDO UM PAINEL PEDINDO USUÁRIO
	E SENHA.
		
		USUÁRIOS PADRÕES COMO user ou admin PODEM SER INSERIDOS NO USUÁRIO E SENHA E CONSEGUIR
		DAR UM BYPASS, TENDO ACESSO AO ROTEADOR. 
		
		OBS: PARA ENTRAR NO PAINEL DO ROTEADOR, PRECISA ESTAR CONECTADO A REDE WIFI DELE.
		
		EXTRA: QUANDO SE ENTRA NO PAINEL NORMALMENTE MOSTRARÁ A MARCA DO ROTEADOR,
		CASO OS USUÁRIOS ADMIN OU USER NÃO DÊ CERTO, PESQUISE NA WEB POR USUÁRIOS PADRÕES
		DESSA MARCA E TESTE.
