O PHARMING É UMA TÉCNICA SIMPLES DE SE ENTENDER NA ENGENHARIA SOCIAL, ELE SE TRATA
DE QUANDO UMA PESSOA ENTRA EM UM SITE LEGÍTIMO, MAS POR UMA VULNERABILIDADE, VOCÊ
CONSEGEU FAZER COM QUE A VITÍMA SEJA REDIRECIONADA A UM WEBSITE SEMELHANTE, MAS CONTENDO 
UM VIRUS PARA INFECTAR OS USUÁRIOS.

	1.ENCONTRAR UMA VULNERABILIDADE EM UM WEBSITE
	QUE POSSA SER USADA PARA O PHARMING
	
	2.CRIAR UM WEBSITE FALSO PARECIDO COM O QUE
	ESTÁ VULNERÁVEL.
	
	3.INJETAR UM MALWARE NO WEBSITE FALSO.
	
	4.USAR O LINK DO WEBSITE LEGÍTIMO 
	PARA INFECTAR PESSOAS.
	
	
	UMA FERRAMENTA QUE PODE SER USADA PARA PHARMING É O BEEF, SE APROVEITANDO DE UMA VULNERABILIDADE XSS.
