Weak Implementation são implementações fracas em aplicações e aqui falarei 2 tipos: o Weak Cadastration Implementation e o Weak Reset Password Implementation.

Weak Cadastration Implementation:

1.Crie uma conta

2.Vá no email de confirmação de email.

3.Copie o Link ou o abra.

4.Veja se está como HTTP://

Weak Reset Password Implementation:

1.Vá em esqueceu sua senha.

2.Vá no email que recebeu para alterar a senha.

3.Copie ou acesse o link.

4.Veja se está como HTTP://


Impacto: Por está em HTTP ao invés de HTTPS significa que se algum agente mal intencionado estiver sniffando a rede, ele poderá ter acesso de forma limpa aos dados como a senha nova por exemplo.
