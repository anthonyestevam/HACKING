O FILE UPLOAD(SHELL UPLOAD) É UMA VULNERABILIDADE QUE OCORRE QUANDO UM SITE TEM ALGUMA ÁREA QUE FAZ UPLOAD DE ARQUIVOS, MAS NÃO É FILTRADO, PODENDO ASSIM MANDAR UM ARQUIVO MALICIOSO.

FORMAS DE ATAQUES:
	
	REVERSE SHELL:
		
		1.CRIAR OU BAIXAR UM SCRIPT DE SHELL REVERSA EM PHP.
		
		2.EDITAR O ARQUIVO E COLOCAR O SEU IP PARA ELE FAZER CONEXÃO
		
		3.ESCUTAR NA PORTA QUE SERÁ FEITA A REVERSE SHELL COM NETCAT.
		
		4.FAZER O UPLOAD DESSE ARQUIVO NO SITE.
		
		5.SE TUDO DER CERTO, VOCÊ VAI ESTAR CONECTADO NA MÁQUINA ALVO.
		
		6.CASO O SITE NÃO ACEITE EXTENSÃO PHP, TEM FORMAS DE CONSEGUIR BYPASS,
		ALTERANDO A EXTENSÃO PHP PARA:
		
				.PHP7 .PHP5 .PHTML .PHP3 .PHP4 .JPG.PHP
				
		EXTRA: PODE PROCURAR POR WEB SHELLS NA INTERNET PARA FAZER UM SHELL UPLOAD, POIS EXISTEM MUITOS 
		BEM COMPLEXOS E PRONTOS PARA USO.
