O GOOP É UMA FERRAMENTA QUE PROCURA GIT EXPOSED, PODENDO USAR ESSA FERRAMENTA PARA PROCURAR EM VARIOS PATH DO MESMO DOMINIO.
	
	COMANDOS: goop site.com -f : PROCURA POR REPOSITORIO GIT EM APENAS UM DOMINIO
		  
		  goop -l urls.txt -f: PROCURA POR REPOSITORIOS GIT EM DIVERSOS DOMINIOS DENTRO DE UM ARQUIVO OU EM DIVERSOS PATH DO MESMO DOMINIO.
		  
		  -f É UM PARÂMETRO QUE FAZ BRUTEFORCE E É INTERESSANTE UTILIZÁ-LO SEMPRE, APESAR
		  DE NÃO SER UM PARÂMETRO OBRIGATÓRIO.
		  
		  OBS: SEMPRE QUE NAS LINHAS APARECER CODE=404, SIGNIFICA QUE NÃO FOI ACHADO NENHUM
		  GITT EXPOSED, MAS SE APARECER DIVERSAS LINHAS SEM O CODE=404 SIGNIFICA QUE FOI   	
		  ENCONTRADO UM GIT EXPOSED E VAI SE TUDO SALVO EM UMA PASTA COM O NOME DO DOMINIO.
		  
		  EXTRA: COMO JA DITO ANTES O GOOP TESTA O .GIT EM DIVERSAS URLs, ENTÃO SERIA INTERESSANTE FAZER UM FUZZING E GUARDAR
EM UM ARQUIVO TODOS O RESULTADO DO FUZZING E JOGAR PARA O GOOP,
PORQUE PODE CONTER O GIT EXPLOSURE EM ALGUM PATH COMO POR EXEMPLO:
https://site.com/v1/usr/.git. ENTÃO É INTERESSANTE TESTAR EM TODAS
URLs COLETADAS DO DOMINIO.
