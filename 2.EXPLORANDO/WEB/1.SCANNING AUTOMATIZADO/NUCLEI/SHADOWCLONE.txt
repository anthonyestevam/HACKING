O SHADOW CLONE SERVE PARA OTIMIZAR TEMPO QUANDO SE TRATA DE SERVIÇOS ENORMES, POIS ELE JOGA A DEMANDA
PARA 80 MAQUINAS FAZEREM E APÓS ISSO RETORNAM O RESULTADO, E É UM ÓTIMO ALIADO DO NUCLEI QUANDO SE
TRATA DE MUITAS URLs PARA ENCONTRAR VULNERABILIDADES.
	
	COMANDO: python3 shadowclone.py -i urls.txt -c 'nuclei -l {INPUT} -severity low,medium,high,critical'
	
	-I É O INPUT OU SEJA O ARQUIVO COM AS URLs NO CASO.
	
	-C É PARA RODAR A FERRAMENTA QUE SERIA O NUCLEI.
	
	OBS: CASO USE TEMPLATES, O SHADOWCLONE SÓ RODA SE FOR COM 
	O TEMPLATE ESPECIFICADO, ELE NÃO RODA PASTAS.
	
	EX: nuclei -t /templates
	
	O SHADOWCLONE NÃO RODARIA.
	
	AGORA: nuclei -t /templates/xss/xss.yaml RODARIA.
