APÓS FAZER COLETA DE SUBDOMINIOS, VOCÊ PODE ENVIAR TODOS OS SUBDOMINIOS COLETADOS PARA O BURP SUITE
PARA FAZER ANÁLISE DE REQUISIÇÕES, TESTAR VULNERABILIDADES, ETC...
	
	1.VÁ NAS CONFIGURAÇÕES DO BURPSUITE E VEJA QUAL O IP E PORTA QUE APARECEM NA ÁREA
	DE PROXY LISTERNES.
	
	2.AGORA COPIE O IP COM A PORTA, QUE PODE SER ALGO PARECIDO COM "127.0.0.1:8080"
	
	3.VÁ NO TERMINAL E USE O HTTPX PARA PASSAR OS SUBDOMINIOS PARA O BURPSUITE USANDO
	O IP E PORTA QUE ESTÃO NO BURP.
		
		COMANDO: cat <subdomains_arquivo> | httpx -proxy http://<ip>:<porta>
		
		EX: cat subdomains.txt | httpx -proxy http://127.0.0.1:8080
		
	4.APÓS ISSO TODOS OS SUBDOMINIOS IRÃO PARA O BURP SUITE.
