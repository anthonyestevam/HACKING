O HTML INJECTION É UMA VULNERABILIDADE SIMPLES EM QUE VOCÊ PODE INSERIR
TAGS DE HTML EM CAMPOS DE INPUT  OU PARÂMETROS DA URL QUE NÃO FORAM HIGIENIZADOS E REFLETEM O INPUT
NA TELA.

EX:
	
	<iframe src="url">
	<h1>oi</h1>
	
PODENDO FAZER TAMBÉM ATAQUES XSS POR MEIO DELE.

PODENDO OCORRER DO SISTEMA ARMAZENAR O CÓDIGO HTML OU SER ALGO REFLETIDO, EM QUE SÓ APAREÇA PARA VOCÊ.

DICA:CASO TENTE DE PRIMEIRA E NÃO DÊ CERTO, TENTE ENCODAR A TAG EM URL ENCODE E TESTE NOVAMENTE.

EXTENSÃO PARA ENCODE E DECODE: URL ENCODE DECODE.
