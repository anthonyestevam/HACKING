O GIT EXPOSED É UMA FALHA SIMPLES, MAS QUE PODE TRAZER GRANDES DANOS.
ELA CONSISTE NO VAZAMENTO DA PASTA GIT, QUE POR SÍ SÓ JA É UMA FALHA,
MAS ALÉM DISSO, DENTRO DO GIT PODE TER DADOS CONFIDENCIAIS, COMO CREDENCIAIS, ETC...

PARA ACHAR ESSA FALHA É BEM SIMPLES COM O USO DE GOOGLE HACKING.
		
		1.PROCURE PELA PASTA GIT.
			
			FORMAS:
				GOOGLE DORKS:	
				
					inurl:"git" site:"site.com"
					intitle:"Index Of" intext:"git" site:"site.com"
					
				DOTGIT: DOTGIT É UMA EXTENSÃO DE NAVEGADORES QUE PROCURA A PASTA GIT NOS SITES
				QUE VOCÊ ACESSA.
				
				DIRECTORY DISCOVER: VOCÊ PODE ENCONTRAR A PASTA GIT, FAZENDO A DESCOBERTA
				DE DIRETÓRIOS USANDO FERRAMENTAS COMO DIRB, DIRBUSTER, GOBUSTER, ETC...
				
				URL: VOCÊ PODE COLOCAR NA URL '/.git'.
					EX:
						 O SITE: seusite.com
						 
						 COLOCAR: seusite.com/.git
						 
		2.APÓS ENCONTRAR A PASTA GIT, VOCÊ PODE BAIXAR A PASTA E TENTAR ENCONTRAR DADOS VAZADOS PELO
		GIT, COMO SENHAS, ETC...
		
		
		
		OBS: SÓ DE ENCONTRAR O GIT JÁ É UMA FALHA, MAS CASO ENCONTRE DADOS DENTRO DO GIT, O NÍVEL DA FALHA
		AUMENTARÁ.
