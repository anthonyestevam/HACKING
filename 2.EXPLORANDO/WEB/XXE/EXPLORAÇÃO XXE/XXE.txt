XXE É UMA VULNERABILIDADE XML EM QUE VOCÊ PODE INTERFERIR NO PROCESSAMENTO DE DADOS E VIZUALIZAR ARQUIVOS DO SISTEMA.

COMO FAZER UM ATAQUE XXE:

	1.IR NA INSPEÇÃO

	2.NETWORK
	
	3.ANALISAR REQUISIÇÃO FEITA
	
	4.IR EM PAYLOAD
	
	5.COPIAR A REQUISIÇÃO DE PAYLOAD
	
	6.CRIAR UM ARQUIVO TXT E COLAR A REQUISIÇÃO
	
	7.ADICIONAR UM CÓDIGO PARA LER OS ARQUIVOS.
		EX: <!DOCTYPE foo[<!ENTITY nomearquivo SYSTEM 'file:///etc/passwd'>]>
		
	8.RENOMEAR O ID DA REQUISIÇÃO PARA O MESMO NOME COLOCADO NO ARQUIVO USANDO '&' ANTES DO
	NOME E ';' DEPOIS.
		EX: &nomearquivo;
	
	9.ENVIAR REQUISIÇÃO PELO TERMINAL:
		curl -d @nomearquivo http://URLDOSITE.com
		
OBS: A URL QUE SERÁ USADA SERÁ A QUE SOLICITA A REQUISIÇÃO. PARA VER A URL QUE SOLICITA A REQUISIÇÃO:
	1.IR EM INSPEÇÃO
	
	2.NETWORK
	
	3.IR NA MESMA REQUISIÇÃO FEITA
	
	4.IR EM HEADERS
	
	5.COPIAR A URL SOLICITADA.
