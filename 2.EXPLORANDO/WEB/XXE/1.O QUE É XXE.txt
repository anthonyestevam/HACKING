O XXE(XML EXTERNAL ENTITY) É UMA VULNERABILIDADE QUE PERMITA QUE VOCÊ INTERFIRA NO PROCESSAMENTO DE DADOS
XML DE UM APLICATIVO. PERMITINDO QUE POSSA VISUALIZAR ARQUIVOS NO SISTEMA DE ARQUIVOS DO SERVIDOR DE APLICATIVOS E INTERAJA COM O SISTEMA BACK-END.

ESSA VULNERABILIDADE OCORRE PORQUE ALGUNS APLICATIVOS USAM O FORMATO XML PARA TRANSMITIR DADOS ENTRE NAVEGADOR E O SERVIDOR. OS APLICATIVOS QUE FAZEM ISSO USAM UMA BIBLIOTECA PADRÃO OU API DE PLATAFORMA
PARA PROCESSAR OS DADOS XML NO SERVIDOR.
