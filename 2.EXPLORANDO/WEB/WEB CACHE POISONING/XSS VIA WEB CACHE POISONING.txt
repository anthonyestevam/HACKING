O XSS PODE SER EXPLORADA PELOS HEADERS DE REQUISIÇÃO E SE O SERVER TIVER VULNERABILIDADE
DE WEB CACHE POISONING, PODE CAUSA UM DANO ENORME.
	
	OS HEADERS Á SEREM EXPLORADOS PODEM SER O REFERER, USER-AGENT, HOST, X-FORWARDED-HOST, X-HOST E HEADERS PERSONALIZADOS
	MAS TEMBÉM PODE SE UTILIZAR DE PARÂMETROS DESCOBERTOS PELO PARAM MINER.
	
	1.INTERCEPTE A REQUISIÇÃO ENVIADA COM O BURP SUITE.
	
	2.ENVIE PARA O REPEATER.
	
	3.TESTE O XSS.
		
		TÉCNICAS:
		
		PAYLOADS NO HOST.
		
		EX: Host:javascript:alert('xss')
		    	
		TESTE TAMBÉM NO REFERER E NO HOST COM O DOMINIO AINDA NELE.
		
		EX: referer: www.site.com/?search=ks
		
		    host: site.com
		
		ALTERE POR EXEMPLO PARA:
			
			referer: www.site.com/?search=<script>alert('xss')</script>
			
			host: site.com"><svg onload=alert('xss')>
			
		VOCÊ TAMBÉM PODE ADICIONAR HEADERS COMO O X-FORWARDED-HOST OU X-HOST
		E TENTAR ADICIONAR O HOST COM UMA CARGA ÚTIL DE XSS
			
			EX: Host: site.com
			    X-Forwarded-Host: evil.com"><svg onload=alert('xss')>
			    
		VOCÊ PODE TAMBÉM TENTAR CRIAR UM HEADER PERSONALIZADO E COLOCAR O PAYLOAD DO XSS NELE.
		
		EX: host:site.com
		    example: <script>alert('xss')</script>
		    
		 PODE SE TESTAR HEADERS QUE FORMA DESCOBERTOS COM O PARAM MINER E ANALISA-LOS
		 PARA TENTAR SOLTAR AS CARGA ÚTEIS NELE DE ACORDO COMO O SERVIDOR INTERPRETA ELE.

	4.USE OS TESTES NO REPEATER E ANÁLISE A RESPONSE PARA VER SE DERAM CÓDIGO
	200, 302 OU 403 E SE FORAM INTERPRETADOS OU SE ENTRARAM EM UMA TAG META.
		
		EXEMPLO DE TAGMETA POLUIDA:
		     <meta property="og:image" content="https://foo."><script>alert(1)</script>"/> 
	
	5.VOLTE PARA O INTERCEPTADOR E ALTERE USANDO A MESMA TÉCNICA QUE POSSA TENHA DADO O CÓDIGO
	200, 302 OU 403.
	
	6.CLIQUE EM 'FORWARD'.
	
	7.DESATIVE O INTERCEPTADOR.
	
	8.VEJA SE NO NAVEGADOR REFLETIU ALGUM POP-UP DO XSS.
	
	9.CASO NÃO TENHA REFLETIDO, REPITA O PROCESSO USANDO OUTRAS TÉCNICAS QUE TENHAM DADO
	CÓDIGO 200, 302 OU 403.
	
	10.CASO TENHA REFLETIDO, A FALHA EXISTE.
	
	EXTRA:SE A FALHA DER CERTO, VOLTE PARA O REPEATER E VEJA A RESPOSTA DA REQUISIÇÃO QUE DEU O
	XSS E PROCURE POR ALGUM LINK QUE TENHA A CARGA DO XSS IMPLANTADA, CASO ACHE, TESTE ELA E SE DER CERTO
	ISSO SERÁ BEM IMPORTANTE PARA A PROVA DE CONCEITO.
