O COMMANDO INJECTION É UMA INJEÇÃO SIMPLES DE SER USADA, NELA VOCÊ USA COMANDOS DE LINUX
EM ÁREAS COMO FORMULÁRIOS,COOKIES OU CABEÇALHOS HTTP ETC... QUE NÃO FORAM FILTRADAS CORRETAMENTE
PERMITINDO ASSIM QUE VOCÊ POSSA INJETAR COMANDOS.
	
	PREENCHA O FORMULARIO E USE ANTES DO COMANDO UM && PARA PODER CONCATENAR 2 COMANDOS
		
		EXEMPLO EM UM FORMULARIO QUE PEDE PARA PINGAR UM IP:
			
			192.168.2.1 && cat /etc/passwd
	EXEMPLO DE COMMAND INJECTION:
		
			cat /etc/passwd: UM COMANDO QUE PODE SER USADO PARA LER INFORMAÇÕES
			DA PASTA PASSWD ONDE TEM INFORMAÇÕES IMPORTANTES.
