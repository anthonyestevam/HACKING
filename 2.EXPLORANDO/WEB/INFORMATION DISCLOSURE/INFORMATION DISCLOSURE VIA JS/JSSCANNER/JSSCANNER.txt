O JSSCANNER É UM SCANNER DE JAVASCRIPT QUE ABRE OS JS E TENTA LOCALIZAR
COISAS COMO TOKENS, SENHAS, ETC... USANDO REGEX, ELE OTIMIZA SEU TEMPO QUANDO SE TRATA
DE ANALISAR MUITOS ARQUIVOS JS.
	
	COMANDO:  python3 JSScanner.py
	
	IRÁ PEDIR O ARQUIVO COM AS URLs DE JS
	E DEPOIS IRÁ PEDIR O ARQUIVO REGEX, DIGITE: regex.txt
	QUANDO PEDIR O ARQUIVO REGEX.
	
	APÓS ISSO ELE IRÁ LER AS URLs JS E CASO ENCONTRE ALGUM PASSWORD OU TOKEN
	ELE IRÁ ALERTAR E IRÁ MOSTRAR A URL DE ONDE ENCONTROU.
	
	CASO ENCONTRE ALGO, VOCÊ PRECISA ENTRAR NA URL E VER O QUE 
	ELE ENCONTROU E TENTAR ENTENDER PARA QUE SERVE E TENTAR USAR
	ESSE DADO COLETADO EM ALGUMA COISA COMO TENTAR POR EXEMPLO USAR
	UM TOKEN PARA TENTAR SE CONECTAR EM CLOUD.
