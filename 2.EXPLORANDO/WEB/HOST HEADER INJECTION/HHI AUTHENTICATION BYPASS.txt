O HOST HEADER INJECTION PODE SE UTILIZADO PARA VOCÊ BURLAR PERMISSÕES, POR EXEMPLO ACESSAR UM PAINEL QUE SÓ PODE ACESSAR
USUÁRIOS QUE ESTÃO NA MÁQUINA DO SERVIDOR.
	
	1.ACHE UM PAINEL QUE SÓ PODE TER ACESSO USUÁRIOS QUE ESTÃO 
	NA MAQUINA DO SERVIDOR.
	
	2.INTERCEPTE A REQUISIÇÃO.
	
	3.ALTERE O HOST HEADER PARA: localhost OU 127.0.0.1
	
	4.ENVIE A REQUISIÇÃO, CASO DÊ CÓDIGO 200, VEJA SE VOCÊ
	CONSEGUIU ACESSAR O PAINEL.
	
	5.FEITO ISSO O ATAQUE FOI BEM SUCEDIDO.
	
	EX: UM PAINEL DE ADMINISTRADOR QUE VOCÊ NÃO TEM PERMISSÃO PARA ACESSAR,
	TENTE FAZER O ATAQUE COM HOST HEADER INJECTION PARA ACESSAR O PAINEL.
