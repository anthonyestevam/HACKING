O ATAQUE CSRF POR WEBSITE FALSO, PODENDO ESSE SITE SER USADO PARA PHISHING CASO O WEBSITE ALVO TENHA
VULNERABILIDADE CSRF.
	
	1.ENTRAR NO CÓDIGO FONTE.
	
	2.COPIAR O FORMULÁRIO DO WEBSITE ALVO.
	
	3.CRIAR UM WEBSITE FALSO.
	
	4.COLAR O FORMULÁRIO NO CORPO DO CÓDIGO HTML DO SEU WEBSITE FALSO.
	
	5.APAGAR A TAG DE BUTÃO DO FORMULÁRIO E A TAG DE LABEL CASO TENHA.
	
	6.ALTERAR O TYPE DA TAG INPUT PARA 'HIDDEN' QUE ASSIM DEIXARÁ ESCONDIDO.
	
	7.COLOCAR O VALUE DO INPUT PARA O QUE DESEJA COMO POR EXEMPLO: PARA SEU EMAIL.
	
	8.COLOCAR A URL ALVO COMPLETA NO ACTION DO FORMULÁRIO.
	
	9.CRIAR UM IFRAME, NOMEAR O IFRAME.
	
	10.CRIAR UMA TAG TARGET NO FORMULÁRIO E COLOCAR NO VALOR DO TARGET O NOME DO IFRAME.
	
	11.CRIAR SCRIPT NO CORPO DO CÓDIGO HTML.
	
	12.USAR O CÓDIGO PARA FAZER A SUBMISSÃO AUTOMATICA DA REQUISIÇÃO.
	
		CÓDIGO: <script>document.forms[0];submit()</script>
		
	EXTRA: PARA MANDAR ESSE WEBSITE PARA QUALQUER VITÍMA, CRIE UM SERVIDOR PYTHON E USE O NGROK
	PARA TRANSFORMAR SEU SERVIDOR INTERNO EM EXTERNO E ASSIM PODER MANDA-LO PARA QUALQER VITÍMA.
