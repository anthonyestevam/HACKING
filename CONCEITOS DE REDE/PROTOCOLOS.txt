PROTOCOLO É UM CONJUNTO DE REGRAS QUE GOVERNA A COMUNICAÇÃO DE DADOS.

	HTTP: É UM PROTOCOLO DE COMUNICAÇÃO QUE TEM TROCA DE DADOS NA WEB. ELE É CONSTRUÍDO EM HTML,CSS ETC. UM DOCUMENTO COMPLETO É RECONSTRUÍDO A PARTIR DOS DIFERENTES SUB-DOCUMENTOS COMO IMAGENS,VIDEOS,TEXTOS ETC. VOCÊ FAZ SOLICITAÇÕES AO SERVIDOR QUE SÃO CHAMADAS DE REQUISIÇÕES(REQUESTS) E O SERVIDOR LHE DÁ RESPOSTAS QUE SÃO CHAMADAS DE RESPONSES. 
	

	HTTPS: O HTTPS TEM O MESMO FUNDAMENTO QUE O HTTP, MAS NO HTTPS TODOS OS DADOS DE REQUISIÇÃO NO SERVIDOR SÃO CRIPTOGRAFADOS. E NECESSITA DE UM CERTIFICADO DIGITAL PARA TESTAR QUE AQUELA COMUNICAÇÃO É AUTENTICA, ASSIM SENDO MAIS 'SEGURO'.
	
	PORTAS UTILIZADAS PELO HTTP/HTTPS: 80, 443
	
	SNMP: ELE É USADO PARA CORREIO ELETRÔNICO. 
	
	PORTA UTILIZADA PELO SNMP: 25
	
	FTP: 
	
	PORTA UTILIZADA PELO FTP: 20, 21
	
	SMTP: UTILIZADO PARA GERÊNCIA DE INTERNET.
	
	DNS: IDENTIFICA ENDEREÇOS IP E ENVIA UMA TABELA COM ENDEREÇOS DE CAMINHO DE ALGUMAS REDES DE INTERNET. ELE É RESPONSÁVEL POR TRADUZIR OS IPs EM NOMES
	
	TCP: ELE OFERECE UM SERVIÇO CONFIAVÉL ENTRE AS APLICAÇÕES.
	
	UDP: ELE ENVIA DADOS SEM PREOCUPAÇÃO QUE OS DADOS CHEGUEM AO DESTINATARIO.
	
	IP: DÁ IDENTIFICAÇÃO AO DISPOSITIVO.
	
	SSH: FAZ ACESSO REMOTO A OUTRA MÁQUINA, MAS CONTÉM CRIPTOGRAFIA.
	
	TELNET: FAZ ACESSO REMOTO A OUTRA MÁQUINA, MAS SEM CRIPTOGRAFIA.
