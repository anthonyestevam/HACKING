A SHELL REVERSA E BIND SHELL É UMA TÉCNICA MUITO IMPORTANTE NO PENTEST E ELA TEM DUAS FUNCIONALIDADES.
	
	1.CONSEGUIR SHELL APARTIR DE UMA VULNERABILIDADE WEB POR EXEMPLO, COMO UM COMMAND INJECTION,
	CONSEGUINDO ASSIM TER ACESSO AO SISTEMA
	
	
	2.MANTER PERSISTÊNCIA EM UMA MÁQUINA, MESMO QUE VOCÊ SAIA DA MÁQUINA, VOCÊ PODE VOLTAR
	A SE CONECTAR NELA QUANDO QUISER NOVAMENTE.
	
	
	
	EXTRA: UM SITE MUITO BOM PARA FAZER BIND SHELL E REVERSE SHELL É O GTFOBINS, ONDE TEM DIVERSOS
	PAYLOADS PARA SHELL REVERSA E BIND SHELL USANDO DIVERSOS TIPOS DE FERRAMENTAS.
