O REMOTE DESKTOP É UM SERVIÇO QUE RODA NA PORTA 3389 E É UTILIZADO PARA PODER OUTROS USUÁRIOS
ACESSAREM O DESKTOP DAQUELE SISTEMA, E VOCÊ PODE CONSEGUIR FAZER ISSO, CASO OBTENHA UMA CREDENCIAL,
TENHA O NOME DO DOMINIO DO ACTIVE DIRECTORY E A PORTA ESTEJA ABERTA.
	
	PARA FAZER ISSO É SIMPLES, SERÁ USADO A FERRAMENTA RDESKTOP.
		
		COMANDO: rdesktop <host> -d <nome)dominio> -u <usuario> -p <senha> 
		
		UM PARÂMETRO BEM INTERESSANTE TAMBÉM É O -r clipboard:CLIPBOARD,
		QUE SERVE PARA QUE VOCÊ POSSA COPIAR E COLAR ARQUIVOS DA SUA MÁQUINA
		PARA A MÁQUINA QUE VOCÊ VAI OBTER ACESSO PELO REMOTE DESKTOP.
		
		NESSE CASO O COMANDO FICARIA DA SEGUINTE FORMA:
			
			rdesktop <host> -d <nome)dominio> -u <usuario> -p <senha> -r clipboard:CLIPBOARD
		 
		 APÓS ISSO, SE TUDO DER CERTO, ABRIRÁ UMA TELA RODANDO O DESKTOP DA MÁQUINA
		 COMO SE FOSSE UMA MÁQUINA VIRTUAL, E ASSIM PODERÁ MEXER NO COMPUTADOR COMO
		 SE ESTIVESSE NELE FISICAMENTE.
		 
		 
		 COM ISSO VOCÊ PODE ACESSAR O POWERSHELL PARA PODER COLETAR INFORMAÇÕES DO 
		 SISTEMA. USANDO COMANDOS COMO:
		 
		 	get-netdomain
		 	
		 	get-netuser
