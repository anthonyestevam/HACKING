UMA FORMA DE CONSEGUIR SHELL TAMBÉM NA PÓS EXPLORAÇÃO DO ACTIVE DIRECTORY É USANDO O PASS-THE-HASH E
PASS-THE-PASSWORD.
	
	PASS-THE-HASH É UMA FORMA DE OBTER A SHELL UTILIZANDO APENAS A HASH DA SENHA E O USUÁRIO
	MAS NEM SEMPRE VOCÊ VAI CONSEGUIR O BYPASS UTILIZANDO APENAS A HASH, ASSIM NESSES CASOS
	VOCÊ PODE TENTAR O PASS-THE-PASSWORD
	
	
	O PASS-THE-PASSWORD É QUANDO VOCÊ DESCRIPTOGRAFA A SENHA E OBTÉM A SHELL ATRAVÉS DO
	USUÁRIO E SENHA.

	APRENDER A UTILIZAR A FERRAMENTA PARA ESSAS FORMAS DE BYPASS EM UM SISTEMA
