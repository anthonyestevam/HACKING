O ACTIVE DIRETCORY É UESM SISTEMA USADO PARA GERENCIAR COMPUTADORES WINDOWS, FAZENDO COM QUE USUÁRIOS DE
UM SISTEMA POSSA ACESSAR OUTROS COMPUTADORES COM O MESMO USUÁRIO, SENDO MUITO UTILIZADO PELAS EMPRESAS.

	O PRIMEIRO PASSO É FAZER A ENUMERAÇÃO DO ACTIVE DIRECTORY.
		
		1.ESCANEAR O IP ALVO COM O NMAP.
			
			COMANDO: nmap -sV <host>
			
			APÓS ISSO VOCÊ PODERÁ VER PORTAS ABERTAS COMO 445, 139 E PODE
			VER A VERSÃO QUE RODA NELAS, E PODE ACHAR SERVIÇOS COMO O LDAP 
			SIGNIFICANDO QUE HÁ O ACTIVE DIRECTORY, NA VERSÃO AS VEZES PODE
			MOSTRAR O NOME DO DOMINIO DO ACTIVE DIRECTORY, SALVE ESSA INFORMAÇÃO
			
		2.FAZER ENUMERAÇÃO DOS SERVIÇOS DESSE IP.
			
			COMANDO: enum4linux <host>
			
			APÓS ENUMERAR, A FERRAMENTA DARÁ MUITAS INFORMAÇÕES COMO O
			NOME DO DOMINIO DO ACTIVE DIRECTORY TAMBÉM, AS VEZES PODE
			DESCOBRIR USUÁRIOS E SENHAS E USUÁRIOS COMUNS
			
		3.CASO TENHA SMB ABERTO, FAÇA UMA ENUMERAÇÃO SMB 
		E CASO ENCONTRE UM USUÁRIO SMB TENTE FAZER A EXPLORAÇÃO COM PSEXEC
		OU USANDO O EVIL TWIN PARA OBTER UMA SHELL.
			
			OBS: MAIS INFORMAÇÕES DO PSEXEC E EVIL TWIN NO DIRETÓRIO
			HACKING/2.EXPLORANDO/REDES/SMB.
