O TUNELAMENTO SSH SERVE PARA DUAS COISAS, PARA ENUMERAR AS VERSÕES DO SERVIÇOS E PARA
FACILITAR NO USO DE EXPLOITS PARA OBTER A SHELL DE MÁQUINAS LOCAIS QUE VOCÊ ENCONTROU
NA ENUMERAÇÃO DE IPs. 
		
	1.APÓS FAZER O ESCANEAMENTO DE PORTAS, ABRA OUTRO TERMINAL, PARA FAZER
	O TUNELAMENTO SSH NA SUA MAQUINA.
	
	2.FAZER O TUNELAMENTO SSH PARA FAZER A ENUMERAÇÃO DAS VERSÕES QUE ESTÃO RODANDO EM
	CADA SERVIÇO,.
		
		COMANDO: ssh -L <porta_abrir>:<host_alvo>:<porta_alvo> <user_root>@<ip_alvo>
		
		EX: ssh -L 777:192.168.2.1:80 root@143.167.23.22
		
		<porta_abrir> É A PORTA QUE SERÁ ABERTA NA SUA MÁQUINA PARA RODAR O SERVIÇO DO IP QUE FOI
		ESCANEADO.
		
		<host_alvo> É O IP DO HOST QUE FOI ENCONTRADO ATIVO NA ENUMERAÇÃO DE IP.
		
		<porta_alvo> É A PORTA QUE FOI ENCONTRADA ABERTA NO IP ENUMERADO.
		
		<user_root> É O USUÁRIO QUE VOCÊ CONSEGUIU ACESSO ROOT.
		
		<ip_alvo> É O IP DA MÁQUINA QUE VOCÊ COMPROMETEU E USOU PARA FAZER A ENUMERAÇÃO DE
		IPs
		
	3.USAR O NETCAT OU O NMAP PARA VER A VERSÃO DO SERVIÇO DA MÁQUINA ALVO.
		
		COMANDO: nc <seu_ip_local> <porta_aberta>
		
		COMANDO: nmap <seu_ip_local> -p <porta_aberta> -sV
		
		
	4.APARECERÁ A VERSÃO DO SERVIÇO QUE ESTÁ SENDO UTILIZADO 
	NA PORTA QUE VOCÊ ESCANEOU E TUNELOU PELO SSH E ALÉM DISSO
	QUANDO FOR USAR EXPLOITS SERÁ DE FORMA MAIS SIMPLES.
	
	OBS: APÓS FAZER O TUNELAMENTO IRÁ ABRIR O TERMINAL DO ALVO, NÃO SAIA DELE,
	SE NÃO O TUNELAMENTO SERÁ DESFEITO, ENTÃO ABRA OUTRO TERMINAL PARA FAZER O
	ESCANEAMENTO DE VERSÃO DE SERVIÇOS.
