PARA QUEBRAR AS SENHAS DE LINUX, PRECISA SE ENTENDER UNS PARÂMETROS PARA QUEBRA-LA
	
	EXEMPLO DE USUÁRIO E SENHA NO /etc/shadow:
		
		root:$6$Pt23lQ$.JSkwalL2820sjM3m/ks:17430:0:92
		
		
	A PALAVRA ANTES DOS PRIMEIROS DOIS PONTOS É O NOME DO USUÁRIO, NESSE CASO SERIA O USUÁRIO ROOT.
	APÓS ESSES DOIS PONTOS E O CIFRÃO TEM UM NÚMERO QUE PODE VARIAR ENTRE 6,5,1 OU 2
	
	SENDO ELAS:
		
		1.CRIPTOGRAFIA MD5
		
		2.CRIPTOGRAFIA BLOWFISH
		
		5.CRIPTOGRAFIA SHA256
		
		6.CRIPTOGRAFIA SHA512
		
	APÓS O SEGUNDO CIFRÃO É O SALT QUE É USADO PARA ALTERAR A HASH DA SENHA.
	
	APÓS O TERCEIRO CIFRÃO SE TEM A HASH DA SENHA, QUE SE ESTENDE ATÉ OS PRÓXIMOS DOIS PONTOS QUE VEM ANTES
	DO NÚMEROS.
