O KALI LINUX WSL É O TERMINAL DO KALI LINUX NO WINDOWS, PODE SE APRENDER A INSTALA-LO NO SITE OFICIAL
DO KALI LINUX.
	
	PARA EXECUTAR O KALI LINUX WSL NO CMD:
			
				COMANDO: kali

	PODE SE EXECUTAR ELE TAMBÉM PELO MENU INICIAR NO APLICATIVO KALI LINUX

	TAMBÉM TEM A INTERFACE DO LINUX QUE PODE SER INSTALADO.

		COMANDO: sudo apt update
			 sudo apt install -y kali-win-kex

	PARA ATIVAR O GUI DO LINUX.
	
	MODO JANELA:

		COMANDO: kex --win -s

	MODO SEAMLESS:
	
		COMANDO: kex --sl -s
